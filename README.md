# 💰 FinanceRAG: Financial Question Answering System

## 📌 Overview

FinanceRAG is a Retrieval-Augmented Generation (RAG) system designed to answer financial queries using domain-specific documents.

## 🚀 Features

* PDF-based knowledge retrieval
* Chunking and embedding using MiniLM
* Vector databases: FAISS & Chroma
* LLMs: GPT2 & DistilGPT2
* Interactive UI using Gradio

## 🧠 Technologies Used

* LangChain
* HuggingFace Transformers
* FAISS / Chroma
* Gradio

## ▶️ How to Run

1. Upload financial PDFs to `finance_pdfs/`
2. Run the notebook in Google Colab
3. Launch the Gradio interface
4. Ask questions

* Assignment 1: RAG System
* Assignment 2: Fine-Tuning GPT2 / DistilGPT2
